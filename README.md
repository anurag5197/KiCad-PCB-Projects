# KiCad PCB Projects — Anurag Kumar Gupta

A collection of PCB designs created in KiCad as part of my 
electronics engineering coursework and self-learning.

## Projects

### Motor Driver — DRV8833
Dual H-bridge motor driver PCB with PWM control, fault monitoring,
and thermally optimized layout using ground planes and thermal vias.

### USB to UART Converter
USB Type-C to UART converter using CP2102 IC for serial communication
between PC and embedded devices. 36 nets, fully routed.

### DC-DC Boost Converter
Boost converter PCB for stepping up DC voltage. Fully routed,
fabrication-ready design.
